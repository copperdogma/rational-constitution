When this command is invoked:
1. Review the active ADR draft for this topic (in `log/ideas/_drafts/`).
2. Summarize the current state of the discussion:
   - Current proposal summary (≤10 bullets)
   - Core Alignment (per Constitutional Core)
   - Pros (≤6)
   - Cons (≤6)
   - Outstanding Questions (enumerated)
   - Recommendation: Reject | Good but needs work | Ready (with rationale)
3. Append the output to the active ADR draft automatically.
4. Respond to the user with the summary and recommendation.
